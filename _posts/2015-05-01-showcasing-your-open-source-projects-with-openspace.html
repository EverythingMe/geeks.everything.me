As a company that bases much of its technology on open source projects, we’ve been contributing and opening up internally built tools ever since we started.

As time passed, we’ve reached a point where our organization on GitHub holds a combination of internal tools, forks of projects we’re using and contributed to and some less interesting bits.

We wanted to share our work with the world, and do it in a way that covers the wide range of disciplines and challenges our team deals with - from Web and Android on the client-side through Python, Go on the server and the Cloud and Analytics tools we’re building and using on a daily basis.

This is where OpenSpace comes in - a simple web app to showcase your GitHub projects.

Look & Feel

When I first laid my eyes on MixItUp by KunkaLabs I knew this is how I wanted our showcase to look and behave, and with the help of @ranbena the great, we’ve created a simple, clean and material’ish looking site.

We’ve color-coded every project based on its name in a consistent way, similar to how we give each contact on your phone a color if there’s no image attached to it on the Quick Contacts panel of our Launcher.

Fetching the data

We’ve used GitHub’s open API to access our organization public repos, filtering out those that are forks and also filtering some that are less relevant. We’ve built a Python tool to do the job so we can later automate the updating of the data rather than manually edit the static json file that powers the site.

Our showcase site is up, if you ever want to showcase your open source projects from GitHub - take OpenSpace for spin and see if it gets the job done.
